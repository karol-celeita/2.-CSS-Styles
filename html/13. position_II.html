z-index

Hay tres ejes, x-index, y-index, z-index(profundidad). el de mayo z-index se muestra primero

  • Para elementos posicionados
  • Para elementos transformados
  • Para elementos grid-items
  • Para elementos flex-items
  • Estructura correcta
    • --z-back :-10; si queremos mandar algo atrás
    • --z-normal :1; si un elemento se comporta anormal
    • --z-tooltip :10; para los tooltips encima de la imagen
    • --z-fixed :100; Elemento encima de todos los demàs
    • --z-modal :1000; Tapa todo , ejemplo modal
  • Si la etiqueta padre tiene z-index, los hijos no podrán estar detrás nunca
