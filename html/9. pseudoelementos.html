Pseudoelementos

::first-line

El elemento first-line reconoce la primera line de cualquier párrafo, al darle inspeccionar no existe el elemento en el DOM (Reduce la pantalla para ver el efecto)

::first-letter

El elemento first-letter, cambia el estilo únicamente a las primeras letras, el first letter, no aplica para elementos inline solo de bloque

::before & ::after

Son elementos que se comportan como hijos al inicio y al final de una etiqueta respectivamente : Por ejemplo para poner los signos de interrogación utilizo before y after (Inspecciona el elemento)
